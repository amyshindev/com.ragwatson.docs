# 🎵 AI 기반 장르 미학 시각화 플랫폼 - 웹사이트 구조 설계도

본 문서는 장르별 미학을 이해하는 원클릭 아티스틱 시각화 툴의 상업화 플랫폼을 위한 10개 페이지 배치 및 사용자 흐름(User Flow) 구조를 다룹니다.

---

## 1. 상단 내비게이션 바 (GNB: Global Navigation Bar)
*사용자가 사이트 어디서든 항상 보게 되는 핵심 네비게이션 영역입니다. 복잡도를 낮추고 핵심 액션에 집중하도록 설계되었습니다.*

*   **좌측 (Logo):** 서비스 브랜드 로고 (클릭 시 메인 화면으로 이동)
*   **중앙 (핵심 메뉴 4개):**
    1.  **둘러보기 (Explore):** 커뮤니티 갤러리로 이동하여 트렌디한 아트웍 탐색
    2.  **기술 소개 (Features):** AI가 음악을 분석하는 원리와 미학적 우수성 증명
    3.  **멤버십 (Pricing):** 요금제 비교 및 구독 유도
    4.  **가이드 (Guide):** 스포티파이 캔버스 등록법 및 FAQ
*   **우측 (액션 버튼):** 
    *   **`[ 스튜디오로 가기 (Studio) ]`** (네온/포인트 컬러 단독 버튼으로 강조)
    *   **유저 프로필 아이콘** (로그인 시 노출, 클릭 시 드롭다운 활성화)

---

## 2. 프로필 우측 드롭다운 메뉴 (Dropdown Menu)
*개인화된 관리 메뉴 및 운영 보조 메뉴를 숨겨두어 UI의 미니멀함을 유지하는 영역입니다.*

*   **마이 아카이브 (My Library):** 내가 만들었던 비주얼 루프 및 업로드한 음원 저장소
*   **구독 관리 (Billing):** 현재 멤버십 상태 확인 및 결제 수단 관리
*   **계정 설정 (Account Settings):** 비밀번호 변경 및 프로필 수정
*   **로그아웃 (Logout)**

---

## 3. 하단 푸터 (Footer)
*사이트 최하단 영역으로, 신뢰도 구축을 위한 법적 규정, 아카이브 콘텐츠, 소셜 미디어 링크를 3~4개의 열로 배치합니다.*

*   **1열 (Brand):** 서비스 로고, 한 줄 가치 제안 카피, 회사 정보 및 연락처
*   **2열 (Product):**
    *   둘러보기 (Explore)
    *   AI 미학 기술 (Features)
    *   스튜디오 (Studio)
*   **3열 (Resources):**
    *   아티스트 쇼케이스 (Artist Spotlight)
    *   캔버스 등록 가이드 (Guide)
    *   저작권 및 라이선스 규정 (Copyright & Licensing)
*   **4열 (Social / Legal):**
    *   소셜 미디어 링크 아이콘 (인스타그램, 디스코드, 유튜브)
    *   이용약관 (Terms of Service)
    *   개인정보처리방침 (Privacy Policy)

## 🗺️ 10개 핵심 페이지 전체 배치 맵 (Page Mapping)

| 페이지 번호 및 이름 | 배치 방식 및 위치 | 기획 및 UX적 배치 목적 |
| :--- | :--- | :--- |
| **1. 메인 화면 (Landing Page)** | 기본 도메인 접속 화면 | 가치 제안 및 비회원도 바로 음원을 업로드할 수 있는 첫인상 제공 |
| **2. 둘러보기 (Explore)** | 상단 GNB 메인 메뉴 / 푸터 | 유저들이 다른 아트웍을 탐색하며 사이트에 머무는 시간(체류시간)을 극대화 |
| **3. 기술 소개 (Features)** | 상단 GNB 메인 메뉴 / 푸터 | 일반 오디오 비주얼라이저와의 차별점을 보여주어 유료 멤버십 결제를 설득 |
| **4. 아티스트 쇼케이스** | 푸터 영역 링크 | 플랫폼을 통해 바이럴된 인디 뮤지션의 스토리를 담아 커뮤니티 신뢰도 확보 |
| **5. 오디오 분석 대시보드** | `[스튜디오]` 페이지 내부 서브 화면 | 음원 업로드 직후 AI가 분석한 음악 유전자 데이터(BPM, 무드 등)를 시각화 |
| **6. 비주얼 스튜디오 (Studio)** | 상단 GNB 우측 강조 버튼 | 서비스의 핵심 목적지이므로 가장 눈에 띄는 독립 버튼으로 유도 |
| **7. 마이 아카이브 (My Library)** | 프로필 드롭다운 메뉴 내부 | 개인의 자산 및 작업 히스토리 성격이 강하므로 개인화 영역에 은닉 |
| **8. 멤버십 (Pricing)** | 상단 GNB 메인 메뉴 / 푸터 | BM 핵심 영역으로 3단 가격 테이블을 통해 무료/유료 혜택을 명확히 대조 |
| **9. 가이드 센터 (Guide)** | 상단 GNB 메인 메뉴 / 푸터 | 스포티파이 캔버스 등 플랫폼별 규격 등록법을 제공해 유저의 실무 이탈을 방지 |
| **10. 저작권 규정 (Licensing)** | 푸터 / 결과물 다운로드 직전 팝업 | 뮤지션들이 안심하고 상업적으로 활용할 수 있도록 저작권 귀속 여부를 명시 |
