# docs — Vault Index

Obsidian vault 루트. **Backend와 Frontend 문서는 서로 wikilink하지 않습니다.**

| Stack | Vault folder | graph-group |
|-------|--------------|-------------|
| Backend | `Backend/` | `claude-backend` |
| Frontend | `Frontend/` | `claude-frontend` |

- Backend 작업 → `Backend/CLAUDE.md` 부터 읽기
- Frontend 작업 → `Frontend/CLAUDE.md` 부터 읽기
- Monorepo 전체 청사진 → repository root `CLAUDE.md` (vault 밖, 링크 없음)

코드 harness는 각 vault 폴더와 `backend/_claude/`, `frontend/_claude/` 하드 링크로 동기화됩니다.
